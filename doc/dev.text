Информация для разработчиков
============================

`DB_Adapter` является открытым и свободным ПО и распространяется по [LGPL][].
В данный момент библиотека находится на стадии разработки, в которой может
принять участие любой желающий.

Для этого достаточно [перейти на github][contribute], сделать `fork`
нашего репозитория и <q cite="http://wikipedia.org/">править смело</q> =).

<a href="#" class="action" onclick="$('#github-howto').toggle(); return false;">Если вы не использовали github</a>

<div id="github-howto" style="display: none">
    <p>Во-первых, как вы уже поняли, мы используем систему контроля версий git.
    Если вы не знаете, как использовать git, мы советуем почитать
    <a href="http://book.git-scm.com/">git community book</a>.</p>

    <p>Сервис github &mdash; это что-то вроде социальной сети для разработчиков,
    построенной на основе git. После регистрации мы рекомендуем ознакомиться
    с <a href="http://help.github.com/">этим документом</a>,
    а затем сделать <a href="http://help.github.com/forking/">форк</a> нашего репозитория.</p>

    <p>Далее вы сможете начать делать правки и передавать
    их нам с помощью механизма <a href="http://help.github.com/pull-requests/">Pull Requests</a></a>
</div>

Кстати, если программировать вам уже надоело, вы можете помочь проекту, если переведёте документацию
на английский язык. Все необходимые файлы есть всё в том же репозитории
(в формате [markdown][]).
